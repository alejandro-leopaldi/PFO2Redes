API Flask - Registro y Login con SQLite

Descripción del proyecto
Este proyecto consiste en una API desarrollada con Flask que permite:
- Registro de usuarios
- Inicio de sesión
- Acceso a una sección protegida (/tareas)

Las contraseñas se almacenan de forma segura utilizando hashing.

Requisitos

- Python 3 instalado
- Visual Studio Code (opcional)
- pip


Instalación

1. Clonar o descargar el repositorio
2. Abrir el proyecto en VS Code
3. Crear entorno virtual:
